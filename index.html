Introduction to CSS

What is CSS?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS stands for Cascading Style Sheet and it is a style sheet language. CSS is what you use to selectively style HTML elements.

Anatomy of a CSS ruleset

The main parts of the ruleset are explained below:

  1. Selector: This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled. To style a different element, you change the selector. There are different types of selctors hghlighted below:
    • Element selector: selects all HTML elements of the specified type.
    • ID selector: selects the element on the page with the specified ID.
    • Class selector: selects the element(s) on the page with the specified class.
    • Attribute selector: selects the element(s) on the page with the specified attribute.
    • Pseudo-class selector: selects the specified element(s), but only when in the specified state.
  2. Declaration: It specifies which of the element's properties you want to style. It is a single rule for example color: red;.
  3. Properties: These are ways in which you can style an HTML element.
  4. Property value: This chooses one out of many possible appearances for a given property.

Important rules to note

  • You need to apply CSS to your HTML document. Otherwise, the styling won't change the appearance of the HTML.
  • Ruleset must be wrapped in curly braces. ({})
  • Semicolon (;) must be used to separate each declaration from the next.

CSS can be used to style HTML elements used to build a webpage. CSS is used to set the style, size, color, position of elements on a page.
